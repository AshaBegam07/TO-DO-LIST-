#  To-Do List App

A simple and efficient To-Do List application built using **React**, **HTML**, **CSS**, and **JavaScript**. This app allows users to manage their daily tasks with a clean and responsive interface.

## Features

- Add new tasks
- Edit and delete tasks

## Technologies Used

- React.js
- HTML5
- CSS3
- JavaScript (ES6)
